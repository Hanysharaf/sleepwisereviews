# SleepWiseReviews Project

## Overview
A sleep product review website built as a static HTML site hosted on GitHub Pages.

## URLs
- **Live Site**: https://hanysharaf.github.io/sleepwisereviews/
- **GitHub Repo**: https://github.com/Hanysharaf/sleepwisereviews

## Tech Stack
- Static HTML with inline CSS
- CSS Grid layouts for responsive design
- JavaScript for interactive elements (hamburger menu, FAQ accordion)
- GitHub Pages for hosting

## Site Structure
- Single-page landing site (`index.html`)
- Dark theme with gold accent colors (`--gold: #D4AF37`)
- Mobile-responsive with hamburger navigation

## Key Sections (11 Essential Elements)
1. Header with logo and navigation
2. Hero section with CTA
3. Social proof (statistics)
4. Media/product showcase
5. Benefits/features grid
6. About section
7. Newsletter signup
8. Testimonials (6 reader reviews)
9. FAQ accordion (6 questions)
10. Final CTA
11. Footer with links
