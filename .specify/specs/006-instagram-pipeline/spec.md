## Content Format

### Captions
- Length: 150–200 words
- Structure: hook + tip/value + CTA ("link in bio")
- Hashtags: 5–10 relevant sleep/product hashtags
- No affiliate links in captions (Constitution Principle II)

### Carousel (per post)
- 5 slides (s1–s5)
- Generated by `build_carousels_batch.py`
- Slide content stored in `automation/data/slide_content.json`

### Single Images
- Generated by DALL-E 3 via `generate_ig_images.py`
- Prompts in `automation/data/ig_image_prompts.txt` and Google Sheets

---

## Queue Structure

`automation/data/instagram_queue.json`:
```json
[
  {
    "caption": "...",
    "hashtags": ["#sleep", "..."],
    "image_url": "...",
    "slide_urls": ["s1_url", "s2_url", "..."],
    "article_url": "https://sleepwisereviews.com/posts/..."
  }
]
```

---

## Google Sheets Integration

- Queue items tracked in content calendar sheet
- Image prompt column → sync via `sync_prompts_to_sheet.py`
- Generated image URLs + QA status pushed back via `update_sheet_urls.py`
