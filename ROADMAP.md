## Analytics Focus

Based on Pinterest data:
- **Weighted Blankets** = 52% of impressions (focus here!)
- **"Best X for Sleep"** format works well
- **Product-focused pins** get more outbound clicks

### Strategy:
1. Create more weighted blanket content
2. Use "Best [Product] for Sleep" titles
3. Include clear Amazon CTA
4. Track which articles get traffic
