## PART 3: Affiliate Link Placement Strategy

**Rule:** Never cluster links — spread them naturally.

**Per article:**
- First link: within first 300 words (intro or quick summary box)
- Product cards: each card has ONE affiliate button
- Comparison table: each row links to product page
- Mid-article callout: one link
- End of article: "Check current prices" section
- Maximum 8 affiliate links per article

**Link format used in all HTML files:**
```
https://www.amazon.com/s?k=PRODUCT+NAME&tag=YOUR-AFFILIATE-TAG
```
Replace `YOUR-AFFILIATE-TAG` with your actual Amazon tag after approval.
