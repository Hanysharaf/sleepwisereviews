# 5 Claude Prompts for Social Media Managers (2026)

## 1. 30-Day Content Calendar Engineer
**مهندس تقويم المحتوى لـ 30 يوماً**

```
Act as my personal content planner. Build a full 30-day posting schedule for [platform], including daily post ideas, recommended posting times based on audience behavior, a mix of formats (carousels, short videos, stories), a strong hook for every post, and a clear CTA. Deliver it in a format I can copy and post immediately.
```

---

## 2. Step-by-Step Validation & Audit Framework
**إطار التحقق والتقييم خطوة بخطوة**

```
For every step in a tutorial or process, define: what success should look like ('you should see X'), common failure signals ('if Y happens, revisit the previous step'), and quick checkpoint questions so users can verify they're on track.
```

---

## 3. Viral Headlines Engine
**محرك العناوين الفيروسية**

```
You're a proven copywriter who's written hooks that pulled millions of views. Study viral content in [niche] and generate 50 customizable hook frameworks. Organize them by curiosity, controversy, storytelling, lists, and bold claims. Each hook should be designed to stop the scroll instantly.
```

---

## 4. Community Growth & Engagement System
**نظام تنمية المجتمع والتفاعل**

```
Act as a community growth specialist. Create a simple but effective engagement system: authentic comment responses, DM openers that spark real conversations, collaboration outreach templates, methods to turn followers into loyal supporters, and a daily engagement routine that takes less than 30 minutes.
```

---

## 5. Data Analyst & Growth Optimizer
**محلل البيانات ومحسن النمو**

```
You're a social media data analyst. Review these metrics [paste data] and explain what's driving results, what's holding performance back, where content gaps exist, ideal posting frequency, and a clear 90-day growth plan with measurable KPIs.
```
