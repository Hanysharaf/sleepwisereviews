# PART 2: STEP-BY-STEP VALIDATION FRAMEWORK

## Content Creation Checkpoints

### Before Posting - Pre-Flight Check
| Step | Success Looks Like | Failure Signal | Fix |
|------|-------------------|----------------|-----|
| Hook Written | First line creates curiosity gap | Starts with "In this post..." | Rewrite with pattern interrupt |
| Visual Ready | Clear, branded, text readable | Blurry, too much text, off-brand colors | Use Canva template |
| CTA Clear | Single specific action | Multiple asks or no ask | Pick ONE action |
| Hashtags Set | 8-15 relevant tags | 30 random tags or 0 tags | Use hashtag sets below |
| Caption Length | 150-300 words for IG | Too short (<50) or too long (>500) | Edit to sweet spot |

### After Posting - 24-Hour Audit
| Metric | Good | Needs Work | Action |
|--------|------|------------|--------|
| Saves | >2% of reach | <1% | Add more actionable tips |
| Shares | >1% of reach | <0.5% | Make more relatable/controversial |
| Comments | >3% of reach | <1% | Add question in caption |
| Follows | >50 per post | <10 | Check bio CTA |

### Weekly Content Audit Questions
1. Did my top post this week have a controversial or curiosity hook? (If no, test more hooks)
2. Did I post at least 2 carousels? (If no, carousels = highest saves)
3. Did I respond to all comments within 2 hours? (If no, set notifications)
4. Did I post to Stories 5+ times? (If no, stories boost algorithm)
5. Did any post go "viral" (10x normal reach)? (If yes, make 3 variations next week)
